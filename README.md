# iBroadcast-desktop

Election wrapper for iBroadcast web player.

[https://media.ibroadcast.com](https://media.ibroadcast.com)

## Support

- Mac ✅
- Windows ✅
- Ubuntu ❌ (TODO)

## Installation

### Prerequisites

- Install [Node LTS](https://nodejs.org/en/download/)

### Dependencies
``` sh
npm install
```

### Build
``` sh
npm run dist
```

Binaries should be in the dist folder.
